# &lt;digi-clock&gt;

Shows a digital clock known from Svelte examples.

## Default

Default look &amp; feel:

<Canvas withToolbar>
  <Story name="Default">
    {`
<digi-clock></digi-clock>
`}
  </Story>
</Canvas>

## No Seconds

Time without seconds:

<Canvas withToolbar>
  <Story name="No Seconds">
    {`
<digi-clock seconds=false></digi-clock>
`}
  </Story>
</Canvas>

## Use CEST TZ

Shows the time in the CEST time zone (+01:00):

<Canvas withToolbar>
  <Story name="Use EST TZ">
    {`
<digi-clock offset=-300></digi-clock>
`}
  </Story>
</Canvas>

## Configurable

Configurable parameters:

<Canvas withToolbar>
  <Story name="Configurable" argTypes={{
    seconds: {
      description: 'disables the second part',
      control: 'boolean',
      table: { defaultValue: { summary: true } }
    },
    offset: {
      description: 'adds the offset in minutes to UTC',
      control: 'number',
      table: { defaultValue: { summary: 0 } }
    },
    onTick: {
      description: 'when the second or the minute hand moves',
      action: 'tick'
    }
  }} args={{
    seconds: false, offset: 60
  }} parameters={{
    controls: { disable: false }
  }}>
    {({ seconds, offset }) => `
<digi-clock seconds=${seconds} offset=${offset}></digi-clock>
`}
  </Story>
</Canvas>
