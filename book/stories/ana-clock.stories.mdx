# &lt;ana-clock&gt;

Showing an analogue clock known from Svelte examples.

## Default

Default look &amp; feel:

<Canvas withToolbar>
  <Story name="Default">
    {`
<ana-clock></ana-clock>
`}
  </Story>
</Canvas>

## Twelve Markers

Show twelwe markers (for five-minute intervals) only:

<Canvas withToolbar>
  <Story name="Twelve Markers">
    {`
<ana-clock markers=twelve></ana-clock>
`}
  </Story>
</Canvas>

## Four Markers

Show four markers (for quoters of hour) only:

<Canvas withToolbar>
  <Story name="Four Markers">
    {`
<ana-clock markers=four></ana-clock>
`}
  </Story>
</Canvas>

## No Markers

Show no markers:

<Canvas withToolbar>
  <Story name="No Markers">
    {`
<ana-clock markers=none></ana-clock>
`}
  </Story>
</Canvas>

## No Second Hand

Show no second hand:

<Canvas withToolbar>
  <Story name="No Second Hand">
    {`
<ana-clock secondhand=false></ana-clock>
`}
  </Story>
</Canvas>

## Use CEST TZ

Shows the time in the CEST time zone (+01:00):

<Canvas withToolbar>
  <Story name="Use CEST TZ">
    {`
<ana-clock offset=60></ana-clock>
`}
  </Story>
</Canvas>
